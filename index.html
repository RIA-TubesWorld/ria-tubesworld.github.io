RIA TubesWorld 入服指南

TubesWorld 是 RIA 旗下的一个以多模组生存为主题的 ModdedMinecraft 服务器。

在开始之前，为了避免不必要的麻烦和保证良好的游戏体验，请再花几分钟耐心读完本指南。如果你是有经验的玩家，可以考虑跳过新手常见问题。

新手常见问题

什么是 Java ？如何安装 Java？什么是 OpenJ9？

JRE (Java Runtime Enviromnent) 是 Java 版 Minecraft 运行必须的环境，因为 Java 版的 Minecraft 必须运行在JVM(Java Virtual Machine) 里。

Java 的安装十分简单，只需要在群文件里找到对应版本的 Java 下载运行安装程序即可。

OpenJ9 是不同于 HotSpot(官方JRE中所包含的JVM) 的一个 JVM，在内存管理方面领先 HotSpot 。经过测试，安装并使用 OpenJ9 能大幅减少客户端的内存占用。

游戏不流畅怎么办？什么是 TPS、FPS ？这分别意味着什么？

游戏不流畅有很多情况，TPS低、FPS低，亦或者是网络延迟高等。

什么是TPS？TPS(Ticks Per Second)是一个衡量服务器运行流畅程度的指标。

在 Minecraft 中，每个实体、可刷新的方块在正常情况下每秒会执行20次更新，也就是 tick 。因此，在正常情况下，每秒会有20次 tick ，也就是 TPS=20。然而，Minecraft 中所有方块，甚至是所有维度的更新(tick)操作都是需要依次运行的，也因此，如果要更新的东西过多，或者某个东西更新耗时太长，都会造成一次更新的时间(Tick Time)超过50毫秒，造成 TPS 下降。在这种情况下，TPS 等于 1000ms 除以更新耗时(ms)。

在 TPS 很低的情况下，玩家可以感受到实体的运动变得缓慢、机器加工时间变长，甚至是吃东西的时间也会变得漫长无比。但 TPS 低并不会影响 FPS 。

FPS(Fresh Per Second)，也叫帧数，是指客户端游戏画面每秒刷新的次数。如果 FPS 很低，会造成游戏卡顿、操作不流畅，但并不会造成机器加工时间变长等问题。FPS 低是由于玩家电脑配置不足引起的，与服务器并没有关系。

如果遇到了 TPS 低的问题，可以联系服务器技术解决。如果遇到了 FPS 低的问题，则应该尝试调整视频设置降低画质。如果客户端顿卡，可能是由于内存不足引起的。如果内存不足，首先应该尽量多分配内存，如果电脑物理内存不足，可以尝试使用 OpenJ9 降低内存占用。

游戏闪退了怎么办？我该如何获得帮助？

游戏崩溃后，会在 .minecraft 目录中的 crash-reports 文件夹内生成一份崩溃报告。玩家遇到崩溃以后，如果无法自行解决，应将对应的崩溃报告上传到粘帖箱内(尽量不要截图或者直接上传文件，更不要直接用文字消息发送崩溃报告)，之后附带崩溃报告的链接在群里寻求解决方案。此外，能描述在崩溃前做了什么是更好的。

粘帖箱的使用

将要分享的文本内容复制到 Content 中，填写 Poster 和验证码，点击 Paste! 。之后从浏览器的地址栏复制地址，此即为该文本的链接。

说明与约定

关于登录模组

本服务器使用 SimpleLogin 作为登录模组。该模组将在客户端启动时随机生成一个密码并保存在 .minecraft/config/simplelogin.cfg 文件中。请不要共享该文件，而在更换整合包时务必把这个文件带上，否则会造成无法登录服务器。

关于客户端

客户端相关文件在群文件的 RTW S3 文件夹内。请注意，客户端是有版本号的，并不是每一次都会有最新的完整客户端下载，届时请下载存在的最新版完整客户端，并根据群公告内的客户端版本升级公告进行升级。

提问与讨论的守则

遇到问题是请尽可能参照新手常见问题中的步骤。也请尽量避免询问客户端等在本手册/群公告中已经注明的问题。

讨论问题请尽量和谐，避免过激的语气。
